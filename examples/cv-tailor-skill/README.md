# CV Tailor Skill

An agent skill that helps students tailor their CV to match a specific job description.

## What It Does

- Analyzes a job description for key requirements
- Compares them against your CV
- Suggests specific rewording, reordering, and additions
- Rewrites sections with improvements applied

## How to Use It

1. Copy the `SKILL.md` file into your project:
   ```
   .github/copilot/skills/cv-tailor/SKILL.md
   ```

2. Open Copilot Chat (`Ctrl+Shift+I`) and paste a job description:
   > "Here's a job posting for a Junior Software Engineer at Microsoft. Can you help me update my CV to match it?"

3. Copilot will use the skill to walk you through tailoring your CV for that role.
