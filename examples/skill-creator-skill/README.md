# Skill Creator Skill

An agent skill that helps you create well-structured agent skills following SKILL.md best practices.

## What It Does

- Guides you through creating a new skill step by step
- Includes best practices for triggers, knowledge, workflows, and rules
- Knows the SKILL.md format and file organization conventions
- Helps avoid common mistakes (too vague, too long, missing workflows)

## How to Use It

1. Copy the `SKILL.md` file into your project:
   ```
   .github/copilot/skills/skill-creator/SKILL.md
   ```

2. Open Copilot Chat and ask:
   > "I want to create a skill that helps me write Dockerfiles. Can you help?"

3. The skill will walk you through drafting the SKILL.md with proper structure.
