## Prerequisites

- Python 3.10+
- An OpenAI API key (for the tool calling example)

## Getting Started

Run the setup script for your OS — it checks for Python and pip, installs them if needed, and installs all dependencies.

**Windows (PowerShell):**

```powershell
.\examples\setup.ps1
```

**macOS / Linux (Shell):**

```bash
bash examples/setup.sh
```

Then pick an example and follow its README.
